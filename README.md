# 🧪 SauceDemo Manual Testing Project

A comprehensive **Manual Testing** project conducted on the **SauceDemo** web application. This repository demonstrates industry-standard QA documentation, including test cases, bug reports, test execution reports, screenshots, and test summary.

---

## 📌 Project Overview

This project focuses on validating the core functionalities of the SauceDemo web application through manual testing. The objective was to verify business-critical features, identify defects, and document testing artifacts following standard Software Quality Assurance (SQA) practices.

**Application:** https://www.saucedemo.com

**Testing Type:** Manual Testing

**Tester:** Md. Naimur Rahman

---

The following modules were tested:
