# 🐞 BUG-003: Add to Cart Button Does Not Work for Some Products

## Bug Information

| Field | Value |
|--------|-------|
| **Bug ID** | BUG-003 |
| **Module** | Shopping Cart |
| **Feature** | Add to Cart |
| **Severity** | High |
| **Priority** | High |
| **Status** | Open |
| **Reported By** | Md. Naimur Rahman |
| **Environment** | Windows 11, Google Chrome (Latest), SauceDemo |

---

## Summary

Some products cannot be added to the shopping cart when the **Add to Cart** button is clicked while using the `problem_user` account.

---

## Preconditions

- Login as `problem_user`

---

## Steps to Reproduce

1. Login using `problem_user`.
2. Navigate to the Products page.
3. Click **Add to Cart** for the affected product.

---

## Expected Result

The selected product should be added to the shopping cart successfully.

---

## Actual Result

The product is not added to the shopping cart.

---

## Reproducibility

Always (100%)
