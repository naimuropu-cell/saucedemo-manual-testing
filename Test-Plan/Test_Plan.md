# 2. Objective

The objective of this testing activity is to verify that the core functionalities of the SauceDemo web application work as expected. The testing focuses on identifying functional defects, validating user workflows, and ensuring a smooth user experience.

### In Scope

- User Login

- Checkout Process
- Checkout Validation
- Logout Functionality

### Out of Scope

- Performance Testing
- Security Testing
- Database Testing
- API Testing
- Automation Testing
